# Mini Globed

Una mod multiplayer semplificata e didattica per Geometry Dash.

Permette di vedere altri giocatori nello stesso livello in tempo reale.

## Come usare

1. Avvia il server Python (`server.py`)
2. Configura l'IP del server nelle impostazioni della mod
3. Entra in un livello e gioca con i tuoi amici!

**Nota**: Questo è un progetto didattico/sperimentale.
